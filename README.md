## Installation

Install dependecies with npm

```bash
  npm install 
```
running the development server
```bash
  npm run dev
```

generating setup

windows msi
```bash
  npm run dist:win
```
mac dmg
```bash
  npm run dist:mac
```
linux appImage
```bash
  npm run dist:linux
```
